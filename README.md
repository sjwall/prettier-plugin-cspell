A [Prettier] plugin to sort the keys of a [CSpell] dictionary file.

Config sorting coming soon!

## Installation

```sh
npm i -D prettier prettier-plugin-cspell
```

Update prettier config

```diff
{
+  "plugins": [
+    "prettier-plugin-cspell"
+  ]
}
```
